11.1 Grid Subsetting
Repository entries that are recognized as grids (i.e, can be read by the netCDF-Java Common Data Model (CDM) grid reader (e.g. netCDF, GRIB, GrADS)) provide an option for creating subsets of the grids that can be downloaded or stored in the repository. From the Data menu, select the Subset Grid menu:
The Grid Subset form allows you to subset the grid by region, time and variable and will output the result to a CF compliant netCDF file.
