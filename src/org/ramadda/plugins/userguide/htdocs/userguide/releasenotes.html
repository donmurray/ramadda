New Features

This release contains many bug fixes and a major redesign of the user interface.

Installing Plugins

Most of the plugins are provided in an allplugins.jar file. This has changed in version 1.5. RAMADDA used to provide the plugins in an allplugins.zip file which contained individual plugins that would be unzipped and written out every time RAMADDA ran. All of the plugin code and resources are now placed into one big jar file.

If you are currently running RAMADDA you should remove all of the existing plugins in your <ramadda home>/plugins directory and install the allplugins.jar and/or the individual plugins. If you have already installed the allplugins.jar (not allplugins.zip) then RAMADDA will will automatically install the new allplugins.jar file. If you are installing RAMADDA for the first time you can opt to install the allplugins.jar during the site installation process.
