Installing Plugins

Most of the plugins are provided in an allplugins.jar file. This has changed from before. RAMADDA used to provide the plugins in an allplugins.zip file which contained individual plugins that would be unzipped and written out every time RAMADDA ran. All of the plugin code and resources are now placed into one big jar file.

If you are currently running RAMADDA you should remove all of the existing plugins in your <ramadda home>/plugins directory and install the allplugins.jar and/or the individual plugins. If you are installing RAMADDA for the first time you can opt to install the plugins during the site installation process.

There are also individual plugins available here including some not in the allplugins collection:

  • RAMADDA users workshop
