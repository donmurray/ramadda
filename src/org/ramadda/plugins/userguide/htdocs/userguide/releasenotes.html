RAMADDA User Guide
  
1 RAMADDA Version 1.4 Release Notes
RAMADDA version 1.4 is now available for download at:
    http://sourceforge.net/projects/ramadda/files/ramadda1.4/

Note: When installing a new version of RAMADDA make sure to delete any existing plugins and reinstall them.

New Features

  • New and improved Wiki text facilites for creating rich content and data presentations.
  • New IDV bundles.xml output handler: NOAA/ESRL/PSD Climate Visualizations
  • Support for ESRI shapefiles, Google Earth KML/KMZ files and GPS data, GPX data.
  • Improved Grid Aggregation facilities. Aggregations can be created for local files without using a harvester. Ensemble aggregations are now supported.

Installing Plugins

Most of the plugins are provided in an allplugins.jar file. This has changed from before. We used to provide the plugins in an allplugins.zip file which contained individual plugins that would be unzipped and written out every time RAMADDA ran. We now place all of the plugin code and resources into one big jar file.

If you are currently running RAMADDA you should remove all of the existing plugins in your <ramadda home>/plugins directory and install the allplugins.jar and/or the individual plugins. If you are installing RAMADDA for the first time you can opt to install the plugins during the site installation process.

There are also individual plugins available here including some not in the allplugins collection:
