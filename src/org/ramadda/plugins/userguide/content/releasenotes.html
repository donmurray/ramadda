[ht::head {RAMADDA Version 1.4 Release Notes}] <%nochildlist%> RAMADDA version 1.4 is now available for download at:
http://sourceforge.net/projects/ramadda/files/ramadda1.4/
Note: When installing a new version of RAMADDA make sure to delete any existing plugins and reinstall them. ==New Features==
  • New and improved Wiki text facilites for creating rich content and data presentations.
  • New IDV bundles.xml output handler: NOAA/ESRL/PSD Climate Visualizations
  • Support for ESRI shapefiles, Google Earth KML/KMZ files and GPS data, GPX data.

Installing Plugins

Most of the plugins are provided in an allplugins.zip file. If you are currently running RAMADDA you should remove the existing plugins in your <ramadda home>/plugins directory and install the allplugins.zip and/or the individual plugins. If you are installing RAMADDA for the first time you can opt to install the plugins during the site installation process.

There are also individual plugins available here including some not in the allplugins collection:

  • NLAS LiDAR data services
  • RAMADDA users workshop
  • Travel planning
  • African Adaptation Program templates
