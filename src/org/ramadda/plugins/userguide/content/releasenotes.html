[ht::head {RAMADDA Version <%ramadda.version%> Release Notes}] <%nochildlist%> RAMADDA version <%ramadda.version%> is now available for download at:
    http://sourceforge.net/projects/ramadda/files/ramadda<%ramadda.version%>/
This release contains many bug fixes and a major redesign of the user interface.
Note: When installing a new version of RAMADDA you also need to install new versions of any plugin files. More information below
Plugins are installed in the <ramadda home>/plugins directory. Most of the plugins are provided in an allplugins.jar file. A number of other plugins are listed below.
Note: This has changed in version 1.5. RAMADDA used to provide the plugins in an allplugins.zip file which contained individual plugins that would be unzipped and written out every time RAMADDA ran. All of the plugin code and resources are now placed into one big jar file.
If you have previously installed allplugins.zip then delete the file and install allplugins.jar. If you have previously installed allplugins.jar then RAMADDA will automatically update the plugun at runtime. If you have other plugins installed then you should update them to the latest versions If you are installing RAMADDA for the first time you can opt to install the allplugins.jar during the site installation process.

There are also individual plugins available here including:

  • Point data plugin
  • NLAS LiDAR plugin
