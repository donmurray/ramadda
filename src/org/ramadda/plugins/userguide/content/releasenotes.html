[ht::head {RAMADDA Version 1.4 Release Notes}] <%nochildlist%> RAMADDA version 1.4 is now available for download at:
http://sourceforge.net/projects/ramadda/files/ramadda1.4/

New Features

  • New and improved Wiki text facilites for creating rich content and data presentations.
  • New IDV bundles.xml output handler: NOAA/ESRL/PSD Climate Visualizations
  • A number of new plugins:
    • Ontology

Installing Plugins

Many of the plugins are now provided in an allplugins.zip file. If you are currently running RAMADDA you should remove the existing plugins in your <ramadda home>/plugins directory and install the allplugins.zip. If you are installing RAMADDA you can opt to install the plugins during the site installation process.

There are also individual plugins available here including some not in the allplugins collection:

  • NLAS LiDAR data services
  • RAMADDA users workshop
  • Travel planning
  • African Adaptation Program templates
