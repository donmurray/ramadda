[ht::head {RAMADDA Version <%ramadda.version%> Release Notes}] <%nochildlist%> RAMADDA version <%ramadda.version%> is now available for download at:
    http://sourceforge.net/projects/ramadda/files/ramadda<%ramadda.version%>/
This release of course contains many bug fixes. But it also features a new display front-end - http://ramadda.org/repository/alias/displays.
You may need to update your plugins. Plugins are installed in the <ramadda home>/plugins directory. Most of the plugins are provided in an allplugins.jar file.

  • If you have previously installed allplugins.zip then delete the file and install allplugins.jar.

  • If you have previously installed allplugins.jar then RAMADDA will update it automatically.

  • If you have the pointplugin.jar installed then delete it. It is now provided in the core.

  • If you have other plugins installed then you should update them to the latest versions

  • If you are installing RAMADDA for the first time you can opt to install the allplugins.jar during the site installation process.

    There are also individual plugins available here including:

    • NLAS LiDAR plugin
