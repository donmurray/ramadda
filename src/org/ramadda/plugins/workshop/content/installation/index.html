[ht::head {Installating RAMADDA under Tomcat}] RAMADDA is available at:
http://www.sourceforge.com/projects/ramadda

We're going to install the repository.war:
http://sourceforge.net/projects/ramadda/files/ramadda1.3b/

Steps performed as root:

  1. Install Tomcat 
    yum -y install tomcat6
  2. Copy RAMADDA war file to webapps directory: 
    cp repository.war /var/lib/tomcat6/webapps
  3. Start Tomcat 
    chkconfig tomcat6 on
service tomcat6 start
[ht::foot]
